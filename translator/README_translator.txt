
Voynich Translator Module (OSF Submission Version)
===================================================

Files:
------
- translate.py               : Core translation logic (EVA → English)
- reverse.py                 : Reverse lookup (English → Voynich)
- voynich_lexicon_expanded.json : Full lexicon with POS and meaning
- run_examples.py            : Example usage script (forward + reverse)

How to Use:
-----------
$ python run_examples.py

Requirements:
-------------
Standard Python 3 (no external libraries required)
